# 🎬 Web Tiket Bioskop

Web pemesanan tiket bioskop sederhana berbasis Python menggunakan Streamlit.

## Fitur
- Login user dengan batas 3x percobaan
- Pilih film & jadwal
- Hitung total harga otomatis
- Diskon 10% untuk member
- Download struk pemesanan

## Cara Install

```bash
pip install streamlit
```

## Cara Jalanin

```bash
streamlit run app.py
```
